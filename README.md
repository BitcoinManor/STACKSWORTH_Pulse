# STACKSWORTH_Pulse
A 5" Raspberry Pi touch dashboard + full Bitcoin node with local StacksWorth SatoNak Hub API and kiosk UI. 1st in the STACKSWORTH Family of Touchscreen Bitcoin Dashboards with the ability to run a full node and control your Bitaxe Farm. With the Matrix, &amp; Spark, our Flagship models it leads before the Edge 7" and Infinty 10"  Bitcoin Dashboards.
